# CNU Mate 개인정보 처리방침

시행일: 2026년 5월 26일  
최종 수정일: 2026년 5월 26일

CNU Mate(이하 "앱")는 충남대학교 LMS 할 일·공지·시간표를 모바일에서 확인하도록 돕는 비공식 학습 보조 앱입니다. 본 문서는 앱이 어떤 정보를 어떻게 처리하는지 설명합니다.

## 1. 수집하는 정보

### 1.1 사용자가 직접 입력하는 정보

- LMS 로그인에 필요한 학번 및 비밀번호
- 저장 위치: 기기 보안 저장소(iOS Keychain / Android EncryptedSharedPreferences)
- 용도: 충남대 LMS WebView 로그인 및 데이터 동기화
- 외부 서버 전송: 앱 운영자의 별도 서버로는 전송하지 않음. LMS 웹사이트 로그인 시에만 해당 정보가 LMS에 전달됨

### 1.2 LMS에서 가져와 기기에 저장하는 정보

- 할 일(과제명, 마감일, 강의명 등), 공지 목록, 강의/시간표 정보
- 공지 상세 HTML 및 첨부 파일(사용자가 오프라인 저장을 선택한 경우)
- 저장 위치: 기기 로컬 저장소(SharedPreferences, 앱 문서 디렉터리)
- 용도: 오프라인 표시, 알림 예약, 홈 화면 위젯

### 1.3 사용 통계 (Firebase Google Analytics)

앱 개선을 위해 아래와 같은 **익명 사용 통계**를 Google Firebase Analytics를 통해 수집할 수 있습니다.

- 앱 실행, 화면 이동(할 일/시간표/공지/설정)
- 동기화 시작·성공·실패(실패 원인은 고정 카테고리만, 예: timeout, login_failed)
- 로그인 성공 여부 및 포털 로그인 옵션 사용 여부
- 설정 상태 스냅샷(알림 ON/OFF, 테마, 자동 동기화 ON/OFF 등. 정확한 시각·과제명 미포함)

**수집하지 않는 정보:** 학번, 비밀번호, 과제/공지/강의 제목 및 본문, LMS 계정 식별 정보

사용자는 앱 **설정 > 개인정보 및 통계**에서 사용 통계 전송을 끌 수 있습니다.

## 2. 정보 이용 목적

- LMS 데이터 동기화 및 앱 기능 제공
- 마감 알림 및 일일 요약 알림 제공
- 앱 안정성·사용성 개선을 위한 통계 분석

## 3. 제3자 제공

- **Google LLC (Firebase Analytics):** 익명 사용 통계 처리. [Google 개인정보 처리방침](https://policies.google.com/privacy)
- **충남대학교 LMS:** 사용자가 입력한 로그인 정보 및 LMS 연동 목적의 데이터 요청

앱 운영자는 별도의 백엔드 서버를 운영하지 않으며, LMS 데이터를 제3자에게 판매하거나 광고 목적으로 제공하지 않습니다.

## 4. 보관 및 파기

- LMS 자격증명: 로그아웃 또는 앱 삭제 시 기기에서 삭제
- LMS 캐시 데이터: 로그아웃 시 삭제. 공지 HTML 캐시는 설정에서 별도 삭제 가능
- Analytics: Google Firebase 정책에 따름

## 5. 이용자의 권리

- 설정에서 사용 통계 전송 ON/OFF
- 로그아웃으로 기기 내 LMS 데이터 및 자격증명 삭제
- 앱 삭제로 기기 내 저장 데이터 제거

## 6. 아동

본 앱은 대학 LMS 사용자를 대상으로 하며, 아동을 대상으로 하지 않습니다.
